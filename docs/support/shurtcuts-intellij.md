# Shurtcuts Intellij

**```CTRL + Y```** (Delete Line)  
**```CTRL + D```** (Duplicate Line)  
**```CTRL + F8```** = Create breakpoint  
**```SHIFT + F9```** = Debug  
**```CTRL + SHIFT + F10```** = Run  
**```ALT + 1```** = Abre e fecha a Aba do Projeto  
**```CTRL + SHIFT + F```** = Procura Arquivos  
**```CTRL + SHIFT + O```** = Update Maven  
**```SHIFT + F6```** = Rename File
